# 决策咨询专报

## 关于促进浙江省高质量充分就业的对策建议

---

**报送单位：浙江省就业政策研究专题组**

**日期：2026年5月**

---

### 一、意义与缘由

党的二十大明确将"促进高质量充分就业"作为新时代就业工作的核心目标。2024年5月，习近平总书记在中央政治局第十四次集体学习时进一步强调，高质量充分就业是新时代新征程就业工作的新定位、新使命。

浙江作为高质量发展建设共同富裕示范区，"扩中提低"、缩小收入差距是核心命题，而高质量充分就业正是实现这一目标的根本支撑。2024年，全省城乡居民收入倍差已从2015年的2.07缩小至1.83（数据来源：浙江省统计局《2024年统计公报》），就业质量提升在其中发挥了关键作用。

然而，形势不容乐观。"十四五"期间，浙江就业人口增幅仅约2%，远低于"十三五"时期10%的增速。2025年三季度，全省就业信心指数降至113.3，创近年新低（数据来源：浙江省发展规划研究院，2025年11月）。就业工作重心正从"有没有工作"向"工作好不好"加速转型。

---

### 二、突出问题

**（一）结构性矛盾突出**

全省求人倍率连续保持在1.6以上，"用工荒"与"就业难"并存。2025届高校毕业生单位就业占比仅72.2%，升学占比升至19.0%，"慢就业""缓就业"趋势明显（数据来源：人社部官网，2025年3月）。数字经济核心产业劳动生产率为全社会平均水平的2.3倍，但"数字+制造"复合型人才严重短缺，供需错配加剧。

**（二）新就业形态质量不高**

全省新就业形态群体近300万人，其中外卖骑手约110万人（数据来源：浙江省发展规划研究院）。这一群体普遍面临社保覆盖不足、职业伤害风险高、技能积累有限等困境，劳动关系模糊化问题亟待破解。

**（三）重点群体压力加大**

2025年全省将80万重点群体纳入就业帮扶（数据来源：浙江省《2025年政府工作报告》）。随着AI技术加速渗透，低技能劳动者岗位替代风险持续上升，大龄劳动者、残疾人等群体就业空间进一步收窄。

---

### 三、对策建议

**（一）构建"产业—就业—人才"联动机制，破解结构性矛盾**

1. 建立重点产业人才需求预测预警制度。由省发改委牵头，会同经信厅、人社厅，按季度发布先进制造业、数字经济、现代服务业人才需求目录，以需求引导人才培养方向。

2. 实施"数字+制造"复合型人才专项培育计划。依托浙江大学、之江实验室等平台，到2027年新增数字高技能人才50万人以上。

3. 深化"一试双证"改革。在现有224批次试点基础上，2026年底前实现先进制造业重点领域全覆盖，打通技能人才评价与使用的制度堵点（数据来源：浙江省社科联"之江策"，2025年9月）。

4. 设立县域数字人才专项补贴，推广"周末工程师""云端专家"等柔性引才模式，缓解县域尤其是山区26县的人才短板。

**（二）提升新就业形态就业质量，构建全周期保障体系**

1. 扩大职业伤害保障覆盖面。2026年底前将保障范围拓展至所有平台从业人员，实现300万新就业形态群体应保尽保。

2. 建立灵活就业人员"技能银行"制度。依托"30分钟职业技能培训圈"提供免费短期培训，累计达标可兑换职业技能等级证书，打破"低技能—低保障"的恶性循环。

3. 引导平台企业优化收入分配结构。建立"基础单价+技能等级溢价+全勤奖励"薪酬体系，推动灵活就业从"拼时长"向"拼技能"转型。

4. 建设灵活就业综合服务平台。整合社保缴纳、技能培训、法律援助、岗位对接等功能，实现"一码通办"（数据来源：浙江省《2025年政府工作报告》）。

**（三）强化重点群体精准帮扶，兜牢就业民生底线**

1. 做实"15分钟就业服务圈"。2026年底前实现全省所有街道和中心镇全覆盖，对零就业家庭实行"一户一策"动态清零。

2. 对离校未就业高校毕业生实行"131"服务——1次职业指导、3次岗位推荐、1次培训机会。每年开发政策性岗位3万个以上。

3. 建立大龄劳动者就业过渡机制。对距退休不足5年的大龄失业人员发放社保补贴，严禁各类招聘中的年龄歧视（数据来源：浙江省人民政府官网，2025年4月）。

4. 深化"浙江无欠薪"行动，将保障范围拓展至新就业形态，建立全省统一的工资支付监测预警平台。

**（四）以创业带动就业倍增，打造"浙里创业"最优生态**

1. 设立省级青年入乡创业基金10亿元。聚焦智慧农业、农村电商、乡村旅游，每年扶持2000个以上青年创业项目，带动就业2万人以上（数据来源：浙江省经济信息中心，2025年4月）。

2. 推广"没有围墙的创业园"模式。在杭州、宁波等城市提供"零租金"孵化工位3万个，降低青年创业门槛。

3. 加大创业融资支持。将个人创业担保贷款上限提高至80万元、小微企业提高至500万元，省财政安排创业担保基金5亿元。

4. 构建"创业陪跑"全链条体系。引入创业导师2000名以上，提供"培训—陪跑—再创业"闭环支持，允许创业失败人员享受失业保险待遇，降低创业试错成本。
